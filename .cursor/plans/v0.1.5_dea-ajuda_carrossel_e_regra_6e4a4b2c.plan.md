# v0.1.5 — Carrossel no DEA Ajuda + regra padrão

## Objetivo

- Aplicar a mesma estratégia de carrossel da Home (campo `hero.backgroundImages` + `HeroBackgroundCarousel`) no hero do **DEA Ajuda**.
- Criar uma regra em `.cursor/rules/` descrevendo o padrão de carrosséis no projeto.

## Implementação

1. **Assets**
   - Copiar as imagens anexas para `public/images/`:
     - `dea-ajuda-carousel-01.png`
     - `dea-ajuda-carousel-02.png`

2. **Conteúdo**
   - Atualizar `specs/content/dea-ajuda.json`:
     - `hero.backgroundImage` → primeira imagem local
     - `hero.backgroundImages` → `[img1, img2]`
     - `hero.overlay` → gradiente padrão para legibilidade

3. **Render**
   - Estender schema `ministryLandingFields.hero` em `lib/specs/types.ts` com:
     - `backgroundImages?: string[]`
     - `overlay?: string`
   - Atualizar `components/ministry-landing/MinistryLandingHero.tsx` para:
     - Renderizar `HeroBackgroundCarousel` quando `backgroundImages.length > 1`
     - Caso contrário manter `BackgroundImageLayer` + `GradientOverlay`

4. **Regra Cursor**
   - Criar `.cursor/rules/corpus-criste-carousels.mdc` com:
     - escopo (hero background), implementação obrigatória (`HeroBackgroundCarousel` + CSS), overlay e checklist

5. **Testes e validação**
   - Atualizar `specs/tests/e2e/dea-ajuda-visual.spec.ts` para esperar `hero-background-carousel`
   - Rodar:
     - `npm run test:specs`
     - `npm run build`
     - `CI=1 npm run test:e2e`
